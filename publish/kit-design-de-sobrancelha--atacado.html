# Descrição do produto

Moldes para sobrancelhas para design e preenchimento de suas sobrancelhas com praticidade. São 2 moldes com desenhos vazados para você encontrar o formato ideal para seu rosto. Possui um paquímero para medir suas sobrancelhas com precisão a fim de deixá-las idênticas. Utilize-os também para preenchimento com henna. Utilize o pincel para escovar os pelo das sobrancelhas.

CARACTERÍSTICAS

- Ideal para modular, e Ajudar a preencher a Sobrancelha, com Henna ou Canetas
- Deixe a sobrancelha perfeita, e ambas com os mesmos desenhos
- Pratico, e simples, alem de fácil
- Tamanho Paquímetro: 8cm
- Tamanho do Pincel / Pente: 10cm
- Acompanha Moldes Vazados

ITENS INCLUSOS

1 Paquímetro
1 Escovinha
3 Moldes
